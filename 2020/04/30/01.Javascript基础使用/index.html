运算符 运算规则 范例 结果
= 赋值 a=5 5

*= 乘后赋值 a=5,a*=2 10
/= 除后赋值 a=5,a/=2 2.5
运算符 运算规则 范例 结果
== 相等 4 == 3 false
！= 不等于 4 ! = 3 true
> 小于 4 > 3 true
< 大于 4 < 3 false
<= 小于等于 4 <= 3 false
>= 大于等于 4 >= 3 true
全等和全不等
‘ == ‘和’ != ‘进行隐式转换将string转换成number，在比较前转换成相同类型(非严格相等)
‘===’和’ !== ‘必须类型相等，数据相等，直接比较(严格相等)
